### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CharlesAE/jotai-demo.git
   ```
2. Install Dependencies
    ```sh
    npm install
    ```
2. Run dev server
    ```sh
    npm run dev
    ```
